# UCloud Asset Inventory Guide

用于规范“登录控制台后，如何帮助用户盘点当前项目/当前账号下资产”。

## 边界

1. **ProjectId 可以协助查询与说明**
2. **PublicKey / PrivateKey 只说明获取入口与步骤，不直接输出明文**
3. 资产盘点以 **当前项目** 为基本单位；如果用户要求“账号下所有资产”，要明确提醒可能涉及多个 Project
4. 如果某类资产当前页面或接口无法确认，必须明确说明，不得编造

## 二、标准盘点顺序

按下面顺序盘点，避免遗漏：

| 顺序 | 盘点模块 | 说明 |
|---|---|---|
| 1 | 当前项目信息 | 先确认当前项目与 ProjectId |
| 2 | 控制台首页资源统计 | 获取快速概览 |
| 3 | 计算资源 | UHost / 轻量 / 裸金属 / 容器 |
| 4 | 网络资源 | EIP / 防火墙 / ULB / VPC |
| 5 | 数据与存储 | 数据库 / Redis / US3 |
| 6 | 安全资源 | UWAF / UDDoS / UHIDS / USC |
| 7 | 企业应用 | 域名 / 证书 / 备案 |
| 8 | 汇总结果 | 输出表格与结论 |

## 三、输出建议

### 资产盘点输出结构

| 字段 | 说明 |
|---|---|
| 项目信息 | 当前项目名称、ProjectId |
| 资源摘要 | 各大类数量摘要 |
| 重点安全资源 | 是否有 UWAF / UDDoS 等 |
| 域名与证书 | 是否已具备上线前置资源 |
| 风险与缺口 | 缺域名、缺证书、缺备案等 |
| 下一步建议 | 基于 UCloud 服务给出动作 |

## 四、失败话术 / 边界话术

### 1. 关于公私钥

- **正确说法：**
  - “您可以前往 `https://console.ucloud.cn/uapi/apikey` 查看 API 密钥入口。”
  - “如果页面要求手机验证码，请您按页面提示手动完成验证。”
- **禁止说法：**
  - 直接输出 PublicKey / PrivateKey 明文

### 2. 关于资产不完整

- “当前已经完成常见资源盘点，但如果您账号下有多个 Project，还需要逐个项目继续核对。”
- “当前项目下暂未看到该类资源；如需确认全账号范围，请继续切换其他项目检查。”
- “当前页面/接口未直接返回该类资产，暂时无法在本轮盘点中确认。”

### 3. 关于未开通资源

- “当前项目下未看到已开通的 UWAF 资源。”
- “当前项目下暂未发现域名/证书资源，如需接入 UWAF，建议先准备域名与证书。”
